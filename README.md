# Hashdog
partial reimplementation of Hashcat with a GUI

# Compiling
## install dependencies

| Distribution   | Binary package | Development package |
|----------------|----------------|----------------------|
| Debian/Ubuntu  | libgtk-4-1     | libgtk-4-dev         |
| Fedora         | gtk4           | gtk4-devel           |

## build
- use the makefile, simply run :
```
make
```
